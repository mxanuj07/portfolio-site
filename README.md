## ✨ Features

- 🎨 **Premium Dark UI** — Glassmorphism cards, neon glow accents, and gradient typography
- ⚡ **Blazing Fast** — Powered by Vite with instant HMR (Hot Module Replacement)
- 🎞️ **Cinematic Animations** — Page-wide Motion/React animations, typewriter effects & scroll-linked transitions
- 🌊 **Neon Trails** — Interactive cursor-following neon light effect
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop
- 🧭 **Smooth Navigation** — Scroll progress bar, animated navbar, floating dock footer
- 📄 **Resume Download** — One-click resume PDF download
- 📬 **Contact Section** — Direct email & social links integrated

## 🚀 Run Locally

**Prerequisites:** [Node.js](https://nodejs.org/) (v18 or higher)

```bash
# 1. Clone the repository
git clone https://github.com/YOUR-USERNAME/Portfolio.git

# 2. Navigate into the project
cd Portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open your browser and visit: **http://localhost:3000**
